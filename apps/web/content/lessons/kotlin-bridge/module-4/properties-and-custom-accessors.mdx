Properties replace many Java **getter/setter** pairs:

```kotlin
var name: String = ""
```

Custom accessors:

```kotlin
var raw: String = ""
    get() = field.trim()
    set(v) {
        field = v.trim()
    }
```

## Backing `field`

Use **`field`** only inside accessors for the same property — not arbitrary methods.

<Quiz data={quizzes.midQ} label="Properties" />

## Recap

- **Properties** are first-class; accessors compile to JVM methods (`getX`/`setX`) for Java interop.
- **`field`** references the stored slot without recursion.
