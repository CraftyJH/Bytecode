**`object`** declares a singleton â€” one instance for the whole program (classloader scoped).

**`companion object`** is a singleton tied to an outer class â€” replaces many Java `static` utilities while still allowing inheritance hooks.

```kotlin
class Factory {
    companion object {
        fun newId(): Long = System.nanoTime()
    }
}
```

## Recap

- **`object`**: module-level singleton.
- **`companion object`**: class-scoped singleton for static-ish members.
