**`sealed class`** closes inheritance: subclasses must appear in the same **module** (with caveats for expected declarations in multiplatform — out of scope here).

```kotlin
sealed class Result
data class Ok(val value: Int) : Result()
data class Err(val message: String) : Result()
```

Exhaustive **`when`**:

```kotlin
fun describe(r: Result): String = when (r) {
    is Ok -> "ok:${r.value}"
    is Err -> "err:${r.message}"
}
```

## Recap

- **Sealed hierarchies** express closed sums of variants — compiler-assisted completeness checking.
- Pair with **`when`** for readable domain-specific control flow.
