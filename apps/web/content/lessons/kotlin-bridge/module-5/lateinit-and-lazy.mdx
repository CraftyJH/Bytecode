**`lateinit`** delays assignment for non-null **reference** types:

```kotlin
lateinit var service: PaymentService
```

**`lazy`** delegates compute a value on first access:

```kotlin
private val schema: JsonSchema by lazy { loadSchema() }
```

## Recap

- **`lateinit`**: promise you will assign before read — errors at runtime if broken.
- **`lazy`**: thread-safe (by default) memoization for `val`.
