**`?:`** supplies defaults:

```kotlin
val name = input ?: "unknown"
```

Combine with safe calls:

```kotlin
val country = order?.customer?.address?.country ?: "ZZ"
```

**`let`** executes a block when receiver is non-null:

```kotlin
user?.let { u ->
    println(u.name)
}
```

## Recap

- **`?:`** defaults null away — keep defaults cheap (avoid heavy work on RHS unintentionally).
- **`let`** narrows nullable receivers locally.
