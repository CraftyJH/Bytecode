Java calls Kotlin via **generated class names**:

```java
// Kotlin: fun sanitize(input: String): String = ...
import pkg.FileKt;

String out = FileKt.sanitize(in);
```

**`@JvmStatic`** puts methods where Java expects **`static`** fields.

**Companion object** members surface under **`Companion`** unless annotated.

## Recap

- **`UtilKt`** default for top-levels — tune with **`@file:JvmName`**.
- **`@JvmStatic` / `@JvmOverloads`** shape Java-friendly surfaces.
