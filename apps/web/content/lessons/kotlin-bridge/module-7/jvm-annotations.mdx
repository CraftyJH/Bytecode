Key annotations for JVM interop:

- **`@file:JvmName`**: rename file facade for Java.
- **`@JvmName`**: method/property Java name distinct from Kotlin name.
- **`@JvmOverloads`**: generate Java overloads for default parameters.
- **`@JvmField`**: expose `val` as public field to avoid getter.
- **`@Throws`**: document checked exceptions to Java callers.

## Recap

- Treat annotations as **API design** for Java consumers.
- Combine with **package structure** to keep binary compatibility.
