Collection operations parallel Java **Stream** API but are **stdlib defaults** for Kotlin:

```kotlin
val evens = nums.filter { it % 2 == 0 }
val squares = nums.map { it * it }
val pairs: List<Char> = words.flatMap { it.toList() }
```
