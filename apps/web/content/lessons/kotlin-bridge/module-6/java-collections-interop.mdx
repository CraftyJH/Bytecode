Kotlin **maps Java collections** to Kotlin interfaces with lightweight adapters.

```kotlin
val javaList: java.util.List<String> = ArrayList()
val kotlinList: MutableList<String> = javaList
```

Read-only views (`List`) still alias mutable backing stores unless you copy.

## Recap

- Treat Java-returned lists as **potentially mutable** unless documented otherwise.
- **`toList()`** snapshots when you need independence.
